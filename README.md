# Sovereign MeshPoint Project

## Purpose
Boost Wi‑Fi range and coverage while keeping VPN speed intact.  
Built with <$200 in hardware, this project demonstrates resourceful networking and secure extension.

## Methods
1. **Prototype** — Laptop + Wi‑Fi adapters + VPN app.  
2. **Validation** — Extender linked to VPN‑backed hotspot.  
3. **Scale** — Tested across different cities/states.  
4. **Documentation** — Logs, screenshots, and narrative reflection.

## Results
- Stable VPN‑backed SSID rebroadcast through extender.  
- Lower latency in certain video games when connected to local servers.  
- Portable setup: can be unplugged instantly for control.  
- Reliable performance across multiple locations.

## Cost
Total investment: **under $200** in hardware/software.
